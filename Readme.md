# My dotfiles

## Prerequsites

  - Rust
  https://rustup.rs/
  `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
  - `zsh`
  - `oh-my-zsh`
    - starship prompt https://starship.rs/
  - `fzf`
  - `prettierd`
  - `rg` ripgrep

## Install

```sh
$ zsh install.zsh
```
