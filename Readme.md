# My dotfiles

## Prerequsites

  - Rustup
  https://rustup.rs/
  `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
  - `zsh`
  - `oh-my-zsh`
    - starship prompt https://starship.rs/
  - `fzf`
  - `prettierd`
  - `rg` ripgrep
  - `zellij` optional or `tmux`
  - `exa` optional @TODO: switch to `eza`
  - `cargo install --locked flavours`
